Nebrix Lua API Documentation

Complete reference for scripting games in Nebrix using Lua. Build games with powerful 3D objects, physics, GUI, and more.

Introduction

Nebrix uses Lua 5.4 as its scripting language. This API is designed to feel familiar to developers who have used similar game engines while providing powerful features for creating 3D games.

Note: All property names are case-insensitive (e.g., Position and position both work).

Quick Start

Here's a simple example to get you started:

-- Create a red part
local part = Instance.new("Part", workspace)
part.Position = Vector3.new(0, 10, 0)
part.Color = Color3.fromRGB(255, 0, 0)
part.Size = Vector3.new(4, 1, 2)

-- Add physics
part.Anchored = false
part:ApplyImpulse(Vector3.new(0, 50, 0))

-- Update loop
function onUpdate(dt)
    part:Rotate(Vector3.new(0, 1, 0))
end

Vector3

Represents a 3D vector with X, Y, and Z components.

Vector3.new(x: number, y: number, z: number): Vector3

Creates a new Vector3.

x - The X component (default: 0)
y - The Y component (default: 0)
z - The Z component (default: 0)
local position = Vector3.new(10, 5, 0)
local direction = Vector3.new(0, 1, 0)

Constants

Vector3.zero

Shorthand for Vector3.new(0, 0, 0)

Vector3.one

Shorthand for Vector3.new(1, 1, 1)

Properties

Property Type Description
X or x number The X component
Y or y number The Y component
Z or z number The Z component

Vector2

Represents a 2D vector with X and Y components. Used for GUI positioning and sizing.

Vector2.new(x: number, y: number): Vector2

Creates a new Vector2.

local guiPosition = Vector2.new(100, 50)

Constants

  • Vector2.zero - Equals Vector2.new(0, 0)
  • Vector2.one - Equals Vector2.new(1, 1)

Color3

Represents an RGB color with values from 0 to 1.

Color3.new(r: number, g: number, b: number): Color3

Creates a new Color3 with values from 0 to 1.

local white = Color3.new(1, 1, 1)
local black = Color3.new(0, 0, 0)
Color3.fromRGB(r: number, g: number, b: number): Color3

Creates a Color3 from RGB values (0-255).

local red = Color3.fromRGB(255, 0, 0)
local blue = Color3.fromRGB(0, 100, 255)
Color3.fromHSV(h: number, s: number, v: number): Color3

Creates a Color3 from HSV values (0-1).

-- Hue 0.5 (cyan), full saturation and value
local cyan = Color3.fromHSV(0.5, 1, 1)

Properties

  • R or r - Red component (0-1)
  • G or g - Green component (0-1)
  • B or b - Blue component (0-1)

Instance

Base class for all objects in Nebrix.

Instance.new(className: string, parent: Instance?): Instance

Creates a new instance of the specified type.

className - Type: "Part", "Sphere", "Cylinder", "Wedge", "Model", "Folder", "Script", "Frame", "TextLabel", "TextButton"
parent - Optional parent object
local part = Instance.new("Part", workspace)
local model = Instance.new("Model", workspace)
local frame = Instance.new("Frame", game.PlayerGui)

Properties

Property Type Description
Name string The name of the instance
Parent Instance The parent object
ClassName string The type of the instance (read-only)

Methods

IsA(className: string): boolean

Checks if the instance is of the specified type.

if part:IsA("Part") then
    print("This is a part!")
end
FindFirstChild(name: string): Instance?

Returns the first child with the specified name, or nil.

local head = character:FindFirstChild("Head")
WaitForChild(name: string): Instance?

Returns the first child with the specified name (currently same as FindFirstChild).

GetChildren(): {Instance}

Returns an array of all children.

for i, child in pairs(workspace:GetChildren()) do
    print(child.Name)
end

Part

A 3D physical object with shape, position, and physics properties.

Properties

Property Type Description
Position Vector3 World position of the part
Rotation Vector3 Rotation in degrees (X, Y, Z)
Size Vector3 Dimensions of the part
Color Color3 The color of the part
Transparency number Transparency from 0 (opaque) to 1 (invisible)
Anchored boolean If true, the part won't move from physics
CanCollide boolean If true, the part collides with other parts
Mass number The mass of the part for physics
Velocity Vector3 Current velocity vector 
-- Create and configure a part
local part = Instance.new("Part", workspace)
part.Position = Vector3.new(0, 10, 0)
part.Size = Vector3.new(4, 1, 2)
part.Color = Color3.fromRGB(100, 150, 255)
part.Transparency = 0.5
part.Anchored = false
part.CanCollide = true
part.Mass = 10

Physics Methods

SetPosition(position: Vector3)

Instantly sets the position of the part.

part:SetPosition(Vector3.new(10, 5, 0))
Translate(offset: Vector3)

Moves the part by the specified offset.

-- Move the part up by 5 units
part:Translate(Vector3.new(0, 5, 0))
SetRotation(rotation: Vector3)

Sets the rotation in degrees.

part:SetRotation(Vector3.new(0, 45, 0))
LookAt(position: Vector3)

Rotates the part to face the target position.

part:LookAt(Vector3.new(10, 0, 10))
ApplyImpulse(impulse: Vector3)

Applies an instant impulse to the part (impulse / mass = velocity change).

-- Launch the part upward
part:ApplyImpulse(Vector3.new(0, 100, 0))
ApplyForce(force: Vector3)

Applies continuous force over time (uses delta time).

-- Apply upward force in update loop
function onUpdate(dt)
    part:ApplyForce(Vector3.new(0, 50, 0))
end
AddVelocity(velocity: Vector3)

Directly adds to the part's velocity.

part:AddVelocity(Vector3.new(5, 0, 0))
GetVelocity(): Vector3

Returns the current velocity of the part.

local vel = part:GetVelocity()
print("Speed:", vel.Y)
IsGrounded(): boolean

Returns true if the part is touching the ground.

if part:IsGrounded() then
    print("On ground!")
end
GetForwardVector(): Vector3

Returns the forward direction vector based on rotation.

local forward = part:GetForwardVector()
part:Translate(forward)
GetUpVector(): Vector3

Returns the up direction vector (always 0, 1, 0).

GUI Properties

Property Type Description

Position Vector2 X and Y position in pixels
BackgroundColor3 Color3 Background color
Text string Text content (TextLabel/TextButton only)
Visible boolean Whether the element is visible

game

The root of the game hierarchy. Use it to access services.

game:GetService(serviceName: string): Service

Returns the specified service.

local workspace = game:GetService("Workspace")
local lighting = game:GetService("Lighting")
local gui = game:GetService("PlayerGui")

Direct Properties

  • game.Workspace - The 3D world container
  • game.Lighting - Lighting and atmosphere settings
  • game.ScriptService - Container for scripts
  • game.PlayerGui - Container for GUI elements

Workspace

The 3D world where all parts and models exist. Accessible via workspace or game.Workspace.

-- Access workspace
local ws = workspace
-- or
local ws = game:GetService("Workspace")

-- Add parts to workspace
local part = Instance.new("Part", workspace)

Lighting

Controls the lighting, atmosphere, and time of day in your game.

Properties

Property Type Description
TimeOfDay number Time in hours (0-24)
Brightness number Overall brightness multiplier
Ambient Color3 Ambient light color
OutdoorAmbient Color3 Outdoor ambient light color
FogStart number Distance where fog starts
FogEnd number Distance where fog is complete
FogColor Color3 Color of the fog 
local lighting = game.Lighting

-- Set sunset
lighting.TimeOfDay = 18.5
lighting.Brightness = 2
lighting.Ambient = Color3.fromRGB(150, 100, 100)

-- Add fog
lighting.FogStart = 50
lighting.FogEnd = 500
lighting.FogColor = Color3.fromRGB(200, 200, 220)

Methods

SetTimeOfDay(time: number)

Sets the time of day (0-24 hours).

-- Cycle day/night
function onUpdate(dt)
    local lighting = game.Lighting
    lighting:SetTimeOfDay(lighting.TimeOfDay + dt * 0.1)
    if lighting.TimeOfDay > 24 then
        lighting.TimeOfDay = 0
    end
end

Math Extensions

Extended math functions beyond standard Lua math library.

math.clamp(value: number, min: number, max: number): number

Clamps a value between min and max.

local speed = math.clamp(speed, 0, 100)
math.lerp(a: number, b: number, t: number): number

Linear interpolation between a and b.

local pos = math.lerp(0, 100, 0.5) -- Returns 50
math.sign(value: number): number

Returns -1, 0, or 1 depending on the sign.

local dir = math.sign(-5) -- Returns -1
math.round(value: number): number

Rounds to the nearest integer.

local rounded = math.round(4.7) -- Returns 5
math.randomFloat(min: number, max: number): number

Returns a random float between min and max.

local randomSpeed = math.randomFloat(1.0, 5.0)
math.map(value: number, inMin: number, inMax: number, outMin: number, outMax: number): number

Maps a value from one range to another.

-- Map 50 from range 0-100 to range 0-1
local normalized = math.map(50, 0, 100, 0, 1) -- Returns 0.5
math.distance(vec1: Vector3, vec2: Vector3): number

Returns the distance between two Vector3 points.

local dist = math.distance(
    Vector3.new(0, 0, 0),
    Vector3.new(10, 0, 0)
) -- Returns 10

Table Extensions

Additional table manipulation functions.

table.find(tbl: table, value: any): number?

Returns the index of value in the table, or nil.

local fruits = {"apple", "banana", "orange"}
local index = table.find(fruits, "banana") -- Returns 2
table.getn(tbl: table): number

Returns the length of the table.

local count = table.getn({1, 2, 3, 4}) -- Returns 4
table.clear(tbl: table)

Removes all elements from the table.

local list = {1, 2, 3}
table.clear(list) -- list is now {}
table.clone(tbl: table): table

Creates a shallow copy of the table.

local original = {a = 1, b = 2}
local copy = table.clone(original)
table.merge(tbl1: table, tbl2: table): table

Creates a new table with elements from both tables.

local merged = table.merge(
    {a = 1, b = 2},
    {c = 3, d = 4}
) -- {a=1, b=2, c=3, d=4}
table.shuffle(tbl: table): table

Randomly shuffles the table in place and returns it.

local deck = {1, 2, 3, 4, 5}
table.shuffle(deck)

String Extensions

Additional string manipulation functions.

string.split(str: string, separator: string): {string}

Splits a string into an array.

local parts = string.split("hello,world,test", ",")
-- Returns {"hello", "world", "test"}
string.trim(str: string): string

Removes whitespace from both ends.

local clean = string.trim("  hello  ") -- Returns "hello"
string.startswith(str: string, prefix: string): boolean

Checks if string starts with prefix.

if string.startswith(filename, "player_") then
    print("This is a player file")
end
string.endswith(str: string, suffix: string): boolean

Checks if string ends with suffix.

if string.endswith(filename, ".lua") then
    print("This is a Lua file")
end
string.replace(str: string, find: string, replace: string): string

Replaces all occurrences of find with replace.

local text = string.replace("Hello World", "World", "Nebrix")
-- Returns "Hello Nebrix"

Task Library

Functions for managing asynchronous tasks and timing.

task.wait()

Yields execution (currently a placeholder).

wait()

Global alias for task.wait().

Note: The task library is minimal in the current version. More scheduling features are planned for future releases.

Global Functions

print(...)

Outputs messages to the console.

print("Hello, Nebrix!")
print("Position:", part.Position)

warn(message: string)

Outputs a warning message.

warn("This feature is deprecated")

error(message: string)

Throws an error and stops execution.

if not player then
    error("Player not found!")
end

onUpdate(dt: number)

Called every frame if defined in your script.

function onUpdate(dt)
    -- dt is delta time in seconds
    part.Rotation = part.Rotation + Vector3.new(0, 90 * dt, 0)
end

Global Variables

  • script - Reference to the current script object
  • workspace - Shorthand for game.Workspace

Complete Examples

Spinning Part

local part = Instance.new("Part", workspace)
part.Position = Vector3.new(0, 10, 0)
part.Color = Color3.fromRGB(255, 100, 100)
part.Anchored = true

function onUpdate(dt)
    part.Rotation = part.Rotation + Vector3.new(0, 45 * dt, 0)
end

Bouncing Ball

local ball = Instance.new("Sphere", workspace)
ball.Position = Vector3.new(0, 50, 0)
ball.Size = Vector3.new(2, 2, 2)
ball.Color = Color3.fromRGB(100, 150, 255)
ball.Anchored = false
ball.CanCollide = true

-- Add bounce on collision
function onUpdate(dt)
    if ball:IsGrounded() then
        local vel = ball:GetVelocity()
        if vel.Y < 0 then
            ball:ApplyImpulse(Vector3.new(0, 80, 0))
        end
    end
end

Simple Button Counter

local count = 0

local frame = Instance.new("Frame", game.PlayerGui)
frame.Size = Vector2.new(200, 100)
frame.Position = Vector2.new(10, 10)
frame.BackgroundColor3 = Color3.fromRGB(40, 40, 40)

local label = Instance.new("TextLabel", frame)
label.Size = Vector2.new(200, 50)
label.Text = "Count: 0"
label.BackgroundColor3 = Color3.fromRGB(60, 60, 60)

local button = Instance.new("TextButton", frame)
button.Size = Vector2.new(200, 40)
button.Position = Vector2.new(0, 55)
button.Text = "Click Me!"
button.BackgroundColor3 = Color3.fromRGB(225, 29, 72)

button.Clicked = function()
    count = count + 1
    label.Text = "Count: " .. count
end

Day/Night Cycle

local lighting = game.Lighting
lighting.TimeOfDay = 12

function onUpdate(dt)
    -- Advance time by 0.1 hours per second
    lighting.TimeOfDay = lighting.TimeOfDay + dt * 0.1
    
    if lighting.TimeOfDay >= 24 then
        lighting.TimeOfDay = 0
    end
    
    -- Adjust brightness based on time
    local time = lighting.TimeOfDay
    if time > 6 and time < 18 then
        lighting.Brightness = 2 -- Day
    else
        lighting.Brightness = 0.3 -- Night
    end
end
