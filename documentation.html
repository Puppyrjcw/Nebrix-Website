Welcome to the official Nebrix V0.25 technical manual. This guide covers the engine architecture, API references, and server-side implementation details.

Note: Nebrix V0.25 features a complete interface overhaul. Ensure your scripts are compatible with the new Slint UI integration.

Vector3

The Vector3 class is used for 3D coordinates and direction vectors. All physics calculations in Nebrix utilize this type.

-- Example: Creating a new position
local PART_POS = Vector3.new(15, 0, 0)
print(PART_POS.X) -- Outputs 15

PhysicsService

Handles the global simulation properties, including gravity and N-body calculations.

-- Set global gravity
PhysicsService.Gravity = Vector3.new(0, -196.2, 0)
